

# What:
#### what is an AI?
AI (Artificial Intelligence) is technology that allows computers and machines to simulate human intelligence. Instead of just following a strict set of pre-written rules, AI systems can "learn" from data to perform tasks like recognizing faces, translating languages, or writing code

#### Generative AI (GenAI): 
AI that can create new content (text, images, music, or code) rather than just analyzing existing data.

#### Agentic AI: 
chatbots to autonomous AI agents that can plan multi-step tasks, access external systems, and execute complex workflows (e.g., booking a trip or managing a supply chain) without constant human prompting.

#### what are LLM?	
LLM (Large Language Model) is a type of Artificial Intelligence trained to understand, generate, and manipulate human language

#### what are the types of LLM Models?
Multimodal Reasoning: Interpreting text, images, diagrams, and video (e.g., Gemini, GPT-4o, Grok-1.5V).
Long Context Processing: Analyzing up to 1M+ tokens in a single prompt (e.g., Qwen3, Gemini)
Enterprise Integration: RAG, secure on-premise deployment, and fine-tuning (e.g., Cohere, Mistral).
Agentic Workflows: Tools allowing LLMs to take action, browse the web, and use software (e.g., AutoGen, LangGraph)
