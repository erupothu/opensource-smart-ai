
# LM Studio
LM Studio is a desktop application designed to let you discover, download, and run Large Language Models (LLMs) locally on your own computer. Unlike cloud-based AI services like ChatGPT, it operates entirely offline, ensuring that your data stays private and never leaves your machine

#### what is LM studio?

#### LM Studio installtion.
- Terminal: curl -fsSL https://lmstudio.ai/install.sh | bash
- Download Desktop App: https://lmstudio.ai/
- By default, it runs on http://localhost:1234

#### Terminal commands
- lms status
- lms ls
- lms ps
- lms get <model-name>
- lms load <model-name>
- lms unload --all
- lms server start / stop

- Cmd/Ctrl + Shift + M
- Cmd/Ctrl + N
